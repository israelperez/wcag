Primary Colour Palette

The healthdirect colour palette is a primary expression of the brand and should always be used within all marketing and brand communications.

Referencing contemporary Australian culture the healthdirect colour palette is warm, welcoming and approachable. The broad spectrum of complementary colours permits adaptability and flexibility across both digital and printed environments.

Secondary Colour Palettes

The Secondary Palette should be used as a guide for the tone of a color. We intend for the secondary palette to be ever evolving, offering an endless palette to choose from. The main use of the secondary palette is in the creation of icons and illustrations that will be used across the board to help demonstrate the core benefits of HDA products. We will also be using the secondary palette to create unique secondary brands for each one of HDA's products. Percentage tints can be used of any of these colours.
